<h3>Módulo Concluído: React Development Essentials</h3>
<p>Este repositório reflete os aprendizados e projetos desenvolvidos durante o módulo de React do curso <a href="https://aulas.devclub.com.br/courses">DevClub</a>, sob a mentoria do Rodolfo Mori. O foco foi na construção de Single Page Applications (SPAs) eficientes e na compreensão dos conceitos centrais da biblioteca.</p>
<br>
<div>
  <h3>:trophy: Projeto Destaque: Cadastro de Usuários (CRUD)</h3>
  <p>Como projeto final, desenvolvi um <b>Sistema de Cadastro de Usuários</b> aplicando os conceitos de CRUD. Esta aplicação demonstrou a proficiência em:</p>
  <p>Criação (Create): Formulário para cadastrar usuários (Nome, Idade e E-mail) utilizando useState.</p>
  <p>Visualização (Read): Exibição da lista de usuários cadastrados, gerenciando o ciclo de vida dos dados com useEffect.</p>
  <p>Exclusão (Delete): Funcionalidade para excluir usuários, manipulando o estado global da aplicação.</p>
</div>
<br>
<div>
<h4>:rocket: Tecnologias e Conceitos Dominados:</h4>
<p>Setup Inicial: <b>React, Yarn, Vite</b> (bundler)</p>
<p>Componentização: <b>JSX,</b> Estrutura de Componente, Propriedades, <b>CHILDREN</b></p>
<p>Estado e Ciclo de Vida: useState (Gerenciamento de estado), useEffect (Side effects)</p>
<p>Interatividade: <b>useRef</b> (Referências diretas ao DOM)</p>
<p>Comunicação de Dados: Consumo de <b>APIs</b>, Tratamento de <b>CORS</b>, import e export</p>
<p>Roteamento: Rotas e Navegação com <b> React Router DOM</b></p>
<p>Qualidade de Código: PropTypes (Validação de props)</p>
<p>Estilização: CSS no React, Estilização do CSS pelo JS (Styled Components/CSS-in-JS)</p>
</div>
